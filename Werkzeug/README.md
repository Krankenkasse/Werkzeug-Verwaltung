# Werkzeug Backend



## Table of Contents

- [Installation](#installation)
- [Usage](#usage)

## Installation

To install the necessary dependencies, run:

```bash
npm install
```

## Usage

To start the server, use:

```bash
npm start
```

## Dependencies

The project relies on the following dependencies:

- `basic-auth`: ^2.0.1
- `body-parser`: ^1.20.3
- `dotenv`: ^16.4.5
- `ejs`: ^3.1.10
- `express`: ^4.21.1
- `mariadb`: ^3.3.2
- `multer`: ^1.4.5-lts.1
- `qrcode`: ^1.5.4
